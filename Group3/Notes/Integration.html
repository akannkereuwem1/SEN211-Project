APPLICATION OF DIFFERENTIATION

_Integration of a Function_ Integration is the process of finding the antiderivative of a function, which represents the area under the curve of the function. Types of Integration: - Indefinite Integration: finding the antiderivative of a function - Definite Integration: finding the area under the curve of a function between two points Basic Rules of Integration: - Constant Multiple Rule: ∫af(x)dx = a∫f(x)dx - Sum Rule: ∫f(x) + g(x)dx = ∫f(x)dx + ∫g(x)dx - Power Rule: ∫x^n dx = (x^(n+1))/(n+1) + C - Exponential Rule: ∫e^x dx = e^x + C Integration of Trigonometric Functions: - ∫sin(x) dx = -cos(x) + C - ∫cos(x) dx = sin(x) + C - ∫tan(x) dx = -ln|cos(x)| + C Integration by Substitution: - Used to integrate functions that can be expressed as a composite function - Formula: ∫f(g(x))g'(x)dx = ∫f(u)du Integration by Parts: - Used to integrate functions that are products of two functions - Formula: ∫u dv = uv - ∫v du Definite Integration: - Used to find the area under the curve of a function between two points - Formula: ∫[a, b] f(x) dx = F(b) - F(a) Examples: - Find the indefinite integral of f(x) = 3x^2 + 2x - 5. - Find the definite integral of f(x) = x^2 from x = 0 to x = 2. Tips for Working with Integration: - Use the basic rules of integration to simplify complex integrals. - Apply integration by substitution and integration by parts to solve integrals. - Use definite integration to find the area under the curve of a function.
