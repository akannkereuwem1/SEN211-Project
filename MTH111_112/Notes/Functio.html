FUNCTIONS

A function is a relation between a set of inputs, called the domain, and a set of possible outputs, called the range.
It is a way of describing a relationship between variables. Types of Functions: One-to-One Function: A function where each input corresponds to exactly one output. Many-to-One Function: A function where multiple inputs can correspond to the same output. Onto Function: A function where every output in the range corresponds to at least one input in the domain. Domain and Range: Domain: The set of all possible input values. Range: The set of all possible output values. Function Notation: f(x) is read as "f of x" and represents the output value of the function for the input x. Composition of Functions: The composition of two functions f and g is denoted by (f âˆ˜ g)(x) = f(g(x)). The composition of functions is associative, but not commutative. Inverse Functions: The inverse of a function f is denoted by f^(-1). The inverse function undoes the action of the original function. Graphs of Functions: The graph of a function is a visual representation of the relationship between the input and output values. The graph can be used to determine the domain and range of the function. Examples of Functions: Linear functions: f(x) = mx + b Quadratic functions: f(x) = ax^2 + bx + c Exponential functions: f(x) = a^x Tips for Working with Functions: Use function notation to represent the relationship between variables. Determine the domain and range of the function. Use graphs to visualize the relationship between the input and output values. Use composition and inverse functions to solve problems.
