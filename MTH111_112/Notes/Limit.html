LIMIT AND CONTINUITY

Limit: The limit of a function f(x) as x approaches a is denoted by lim x→a f(x) = L. It represents the value that the function approaches as the input gets arbitrarily close to a. Types of limits: one-sided limit, two-sided limit, infinite limit. Continuity: A function f(x) is continuous at a point a if lim x→a f(x) = f(a). A function is continuous on an interval if it is continuous at every point in the interval. - Types of discontinuity: removable discontinuity, jump discontinuity, infinite discontinuity. Properties of Limits:- Linearity: lim x→a [f(x) + g(x)] = lim x→a f(x) + lim x→a g(x) Homogeneity: lim x→a [cf(x)] = c lim x→a f(x) Sum, difference, product, and quotient rules. Properties of Continuity: Sum, difference, product, and quotient of continuous functions are continuous. Composition of continuous functions is continuous. Continuous functions on a closed interval are bounded and attain their bounds. Types of Discontinuities: Removable discontinuity: lim x→a f(x) exists, but f(a) is not defined. Jump discontinuity: lim x→a- f(x) ≠ lim x→a+ f(x) Infinite discontinuity: lim x→a f(x) = ∞ or -∞ Examples: Find the limit of f(x) = (x^2 - 4) / (x - 2) as x approaches 2. Determine if the function f(x) = |x| is continuous at x = 0. Tips for Working with Limits and Continuity: Use limit properties to simplify complex limits. Check for continuity at a point by using the definition of continuity. Identify types of discontinuities to determine if a function is continuous.
